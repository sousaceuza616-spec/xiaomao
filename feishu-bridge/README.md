# 飞书机器人桥接服务 🐱

这个服务让你可以在飞书里和小猫对话，小猫的消息也会转发到飞书。

---

## 📋 部署步骤

### 1️⃣ 创建飞书应用

1. 打开 https://open.feishu.cn/
2. 登录飞书账号
3. 点击「**企业应用**」→「**创建应用**」
4. 应用名称填「小猫助手」
5. 创建后，在「**凭证与基础信息**」里保存：
   - **App ID** (复制保存)
   - **App Secret** (复制保存)

### 2️⃣ 添加机器人能力

1. 在应用管理页面，点击左侧「**添加应用能力**」
2. 选择「**机器人**」
3. 填写机器人名称（比如「小猫」）和头像
4. 点击「**创建**」

### 3️⃣ 配置权限

1. 点击左侧「**权限管理**」
2. 添加以下权限：
   - `im:message` - 发送和接收消息
   - `im:chat` - 访问群聊信息
3. 点击「**发布版本**」提交审核

### 4️⃣ 配置事件订阅

1. 点击左侧「**事件订阅**」
2. 开启「**启用事件订阅**」
3. **请求地址**先留空（等部署完云函数再填）
4. **订阅事件**：
   - 选择「**接收消息**」→ `im.message.receive_v1`
5. 记下 **Verification Token**（复制保存）
6. 点击「**完成**」

### 5️⃣ 部署到 Vercel

#### 5.1 上传到 GitHub

1. 在 GitHub 创建一个新的仓库（比如 `feishu-bot-bridge`）
2. 把这个项目的所有文件上传上去

#### 5.2 在 Vercel 部署

1. 打开 https://vercel.com/
2. 用 GitHub 账号登录
3. 点击「**Add New Project**」
4. 选择你刚创建的仓库
5. 点击「**Deploy**」

#### 5.3 配置环境变量

部署完成后：

1. 进入项目设置 → **Environment Variables**
2. 添加以下变量：
   - `FEISHU_APP_ID` - 飞书 App ID
   - `FEISHU_APP_SECRET` - 飞书 App Secret
   - `FEISHU_VERIFICATION_TOKEN` - 飞书 Verification Token
3. 重新部署（Settings → Git → Redeploy）

#### 5.4 获取域名

部署成功后，Vercel 会给你一个域名，格式类似：
```
https://feishu-bot-bridge-xxx.vercel.app
```

你的 API 地址是：
```
https://feishu-bot-bridge-xxx.vercel.app/api
```

### 6️⃣ 配置飞书回调地址

1. 回到飞书开放平台的「**事件订阅**」页面
2. 在「**请求地址**」里填入你的 Vercel 域名 + `/api`
   ```
   https://feishu-bot-bridge-xxx.vercel.app/api
   ```
3. 点击「**保存**」
4. 飞书会发送一个验证请求，如果配置正确会显示「验证成功」

### 7️⃣ 发布应用

1. 点击左侧「**版本管理与发布**」
2. 点击「**创建版本**」
3. 填写版本号（比如 1.0.0）
4. 点击「**发布**」

### 8️⃣ 添加机器人到群聊

1. 在飞书里创建一个群（可以只拉你自己）
2. 点击群设置 → **机器人** → **添加机器人**
3. 选择你创建的「小猫助手」
4. 完成！
